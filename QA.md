# Про цикл for 
* Обычная запись цикла `<for (let i = 0; i < name.lenght: i++>` это обычный ход

* Запись цикла в обратную сторону `<for (let i = str.length - 1; i >= 0; i--)>` это реверс. 
Он удаляет индекс с конца и проверяет так же с конца. Поэтому они не смещаются. 

<p>При обычной записи с `++` при удалении индекса остальные смещаются на один и итерация может пропустить нужный нам индекс, потому что шаг следующей итерации будет дальше.</p> 

* <p>Пример: удаляем из массива [1, 2, 3, 4, 5] четные с обычным шагом - удалили 2 с индексом 1 - индексы сместились. Следующая проверка будет по индексу 2, но мы уже сместили все числа на шаг назад по индексам. Следующая проверка индекса 2, по которому сейчас лежит 4, но мы пропустили 3, а это важно, если бы мы должны были работать с нечетными.</p>

</p>Да и в таком массиве это просто везение, что условия задачи выполнены. Добавить еще чисел и конструкция в скором времени посыпется.</p>

<p>Тут открывается реверсивный способ записи цикла for. Когда мы идем от конца <`length - 1>` - индексы при удалении элемента массива смещаются, но с конца - что не является для нас критичным.</p>

* <p>Пример: тот же массив [1, 2, 3, 4, 5] удаляем с шагом с конца с помощью "length - 1".</p> 
* Удаляем с конца: 5 под индексом 4 - пропускаем
* 4 под индексом 3 - удаляем. 5 становится индексом 3
* 3 под индексом 2 - оставляем
* 2 под индексом 1 - удаляем
* 1 под индексом 0 - оставляем

<p>Индексы так же смещаются, но нас это уже не волнует, так как они смещаются позади и шаг не перескакивает никакой индекс</p>

# Про квадратные скобки

* Используются для доступа к элементам `obj[name]` - обращаемся к значению объекта по ключу
* Для обращения к элементу массива по индексу `arr[2]`
* Для получения символа из элемента `str[3]`
<p>Так же можно использовать с методом строк `length` - arr[index].length - получим длину элемента. Если это в цикле - получим длинну каждого элемента на каждом шаге</p>

* Допустимо обращаться к элементу элемента. Если, например, массив в массиве `arr[0][2]` - достанет нулевой индекс массива в массиве, а потом его элемент по 2 индексу. 

# Отличия ![]! от !.!

* Через точку мы обращаемся уже заранее известному (статическому) значению

* Через квадратные скобки - к вычисляемому. По индексу

* После скобок можно использовать спецсимволы. `name[first-name]`. Если записать без скобок через точку "-" будет считаться как операнд минус. Так же с пробелом и прочими символами

* Точка используется для назначения методов массивов. `name.push(), name.filter()` и т.д
