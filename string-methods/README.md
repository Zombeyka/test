# Методы строк

* Метод строки `indexOf()` - показывает начало от индекса элемента (let name = "Anton"; name.indexOf("n") = 1),
* Метод строки `lastIndexOf()` - показывает индекс с конца.

* Метод строки `includes()` - проверяет наличие подстроки. Регистр так же влияет на результат. Возвращает true/false.

* Метод строки `trim()` - убирает пробелы в начале и в конце. Использовать при пользовательском вводе. 

* Метод строки `slice()` - вырезает от i по i, не включая последний индекс. 
console.log(str.slice(0, 5));   // "Hello" (от 0 до 5, не включая 5)
console.log(str.slice(6));      // "World" (с 6 до конца)
console.log(str.slice(-5));     // "World" (5 символов с конца)

* Метод строки `replace()` - заменяет одну подстроку на другую. (data = "bear"; data.replace("bear", "fox") - data = "fox)

# Подстрока, изменяемость/неизменяемость строк

* Кодовое представление символов (char) - `charAt(0)` возвращает символ строки по индексу. `charCodeAt(0)` - возвращает числовое значение для заданного по индексу символа. 

* Превращение строки в массив - `split('')` создаёт из строк массив с указанием разделителя. `Array.from(<имя переменной>)` - так же создаёт массив, но не убирает пунктуационые знаки, если они присутствуют.

# Методы объекта RegExp (регулярное выражение)
* `test()` — проверяет наличие совпадений в строке, возвращает boolean
* `exec()` — ищет совпадение в строке, возвращает массив с информацией о совпадении или null
* `match()` - удовлетворяет ли регулярное выражение запросу (посмотреть повторно)

# Регулярные выражения 
* Синтаксис - `/pattern/flags`. 
* `/передача паттерна/g` - границы регулярного выражения
* `\` - начало передачи паттерна (поведения)
* `g` - флаг (global). Глобальный поиск. Без него замена произойдёт в первом найденном совпадении и break.
* Обычный пробел " "

* Табуляция `\t`

* Перевод строки `\n`

* Возврат каретки `\r`

* Вертикальная табуляция `\v`

* Перевод страницы `\f`

# Пример регулярного выражения от ИИ 

`let phone = "+7 (912) 345-67-89";`

// Без флага g
`console.log(phone.replace(/\s/, ""));`
// "+7(912) 345-67-89" ← заменил только первый пробел!

// С флагом g
`console.log(phone.replace(/\s/g, ""));`
// "+7(912)345-67-89" ← заменил ВСЕ пробелы!
