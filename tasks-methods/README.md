# Метод `.reduce()`

* Метод `reduce()` принимает 2 аргумента в callback `.reduce((аккумулятор, текущий элемент, массив(не об)));`
* Аккумулятор не принимает данные в return - `=>`. Если нужно передать ему конкретное значение - пишем его после стрелочной функции через запятую `=> accumulator + currenValue.key, 0 or {} or []` - этим будет являться аккумулятор. 
* Если не передавать значение, он возьмёт его самостоятельно, взяв за основу первый элемент массива.
Им может быть как число, так и весь объект.

# Метод `.map()`

* Метод `.map()` создаёт новый массив, принимая внесённые изменения в callback 
* Метод не изменяет исходный массив
* Принимает `.map((элемент, индекс, массив(не обязательный)))`

# Метод `.filter()`

* Метод `.filter()` создаёт новый массив, в котором находятся выбранные элементы из исходного. Работает по принципу булевого значения.
* Метод принимает так же 3 аргумента, но обязательный только 1.

# Метод `.forEach()`

* Метод `.forEach()` является более краткой записью цикла for. 
* Метод ничего не возвращает. Для записи данных потребуется переменная
* Так же принимает 3 аргумента, но обязательный только 1.
